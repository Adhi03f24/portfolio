# ADHI03F24 — Extreme Anime Portfolio

A cyberpunk-themed personal portfolio for **Adhithiyan Maliackal** ([@adhi03f24](https://github.com/Adhi03f24)) — AI-Augmented Senior Software Developer, Educator, and Former Ethical Hacker.

Built with React + Vite. Featuring rich animated backgrounds, custom SVG icons, glass-morphism UI, and heavy futuristic card designs throughout.

---

## Live Demo

[adhi03f24.github.io/portfollio](https://adhi03f24.github.io/portfollio/)

---

## Sections

- **Hero** — Cyberpunk intro with animated hex grid, data streams, circuit traces, particle system, and torch reveal
- **Origin Story** — Animated timeline with four career arcs
- **Arsenal** — Skill tree with category filters, S+/S/A/B rank system, and HUD-style stats
- **Projects** — Showcase of work with images and tech tags
- **Services** — AI-Augmented Development, Prompt Engineering, Cybersecurity Training
- **Certifications & Courses** — 127+ certs, 15+ specializations with custom SVG theme icons and unique card designs per provider
- **Dojo** — Teaching philosophy and student testimonials
- **Contact** — Form and social links

---

## Features

- Cyberpunk/anime aesthetic with Orbitron + Exo 2 typography
- Rich animated hero background (perspective grid, hex overlay, data streams, circuit traces, beacon pulses)
- Torch reveal effect on hero character
- CRT scanline, film grain, holographic rings
- Liquid glass header with gradient border
- Custom SVG icons per certification theme (no emoji)
- Unique card styles per provider (Microsoft, Google, Cisco, IBM, etc.)
- Sorted and filterable certification grid
- Smooth Framer Motion animations throughout
- Fully responsive

---

## Tech Stack

- **React 18** — Component architecture
- **Vite** — Fast dev server and build
- **Framer Motion** — Animations and scroll-triggered reveals
- **CSS3** — Custom properties, gradients, masks, clip-paths, keyframes
- **Font Awesome** — Social icons

---

## Run Locally

```bash
git clone https://github.com/Adhi03f24/portfollio.git
cd portfollio
npm install
npm run dev
```

Open **http://localhost:5173** in your browser.

### Build for Production

```bash
npm run build
```

## Project Structure

```
src/
  components/     Header, Intro, ScrollSection
  data/           profile.js, courses.js, skills.js, testimonials.js
  pages/
    Hero/         Hero section (character, torch, background layers)
    Story/        Origin timeline
    About/        Bio, photo, skill tree
    Project/      Project cards
    Services/     Service cards
    Courses/      Certifications grid with SVG icons
    Dojo/         Teaching section + testimonials
    Contact/      Contact form
    Footer/       Footer with socials
    Admin/        Admin panel
  styles/         global.css (theme variables, base styles)
public/           Character images, favicon
```
